# Automated-DIBS
## About
Dibs.py is a script to reserve study rooms on campus at Tennessee Tech.

## Requirements
- Python 3.6+
- Requests module
- .ini config file with user data in _"tokens/"_

## Usage
### Linux/macOS
```console
foo@bar:~$ ./Dibs.py 
```
### Windows
```console
C:\Users\jsmith> python Dibs.py
```
